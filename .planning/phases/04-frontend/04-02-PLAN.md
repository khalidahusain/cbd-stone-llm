<objective>
Build the chat panel with message list, input box, typing indicator, and inline confirmation button. The chat panel occupies the left ~2/3 of the screen and is the primary interaction surface.

Purpose: This is how clinicians interact with the system — they type patient descriptions and see extraction results, follow-up questions, and confirmation summaries. The typing indicator provides feedback during API calls. The confirm button enables the Phase 3 confirmation flow from the UI.

Output: Complete ChatPanel component with all sub-components, connected to ChatContext for state and API calls.
</objective>
