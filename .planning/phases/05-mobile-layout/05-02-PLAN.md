  <action>
Modify `frontend/src/App.tsx` to add responsive behavior. The goal: on mobile (<768px) show a single-column stacked layout with MobileDashboard above ChatPanel; on desktop (>=768px) show the current side-by-side layout unchanged.

Changes to make:

1. **Add import** at top:
   ```typescript
   import MobileDashboard from "./components/MobileDashboard";
   ```

2. **Modify the main content container** (currently `flex flex-1 min-h-0`):
   Change to: `flex flex-col md:flex-row flex-1 min-h-0`
   - `flex-col` = mobile stacked (per D-01, D-06)
   - `md:flex-row` = desktop side-by-side (current behavior preserved)

3. **Add MobileDashboard above chat, visible only on mobile:**
   Inside the main content container, BEFORE the ChatPanel wrapper, add:
   ```tsx
   {/* Mobile dashboard: above chat, hidden on desktop (per D-03, D-04) */}
   <div className="md:hidden shrink-0">
     <MobileDashboard />
   </div>
   ```
   - `md:hidden` ensures it never appears on desktop (D-01).
   - `shrink-0` prevents it from being compressed by flex.
   - MobileDashboard itself returns null when prediction is null (D-03), so nothing shows initially.

4. **Modify ChatPanel wrapper:**
   Change from: `className="w-2/3 border-r border-gray-200"`
   Change to: `className="w-full md:w-2/3 flex-1 md:flex-none md:border-r border-gray-200 min-h-0"`
   - `w-full` = mobile full width (D-02)
   - `md:w-2/3` = desktop 2/3 width (existing behavior)
   - `flex-1` = on mobile, chat fills remaining vertical space after MobileDashboard
   - `md:flex-none` = on desktop, size is set by w-2/3 not flex
   - `min-h-0` = allows flex child to scroll properly

5. **Modify DashboardPanel wrapper:**
   Change from: `className="w-1/3 bg-white"`
   Change to: `className="hidden md:block w-1/3 bg-white"`
   - `hidden md:block` = invisible on mobile, visible on desktop (D-03)
   - Desktop behavior is completely unchanged — same w-1/3, same bg-white

The final App.tsx render should look like:
```tsx
<ChatProvider>
  <div className="flex flex-col h-screen bg-gray-50">
    <header className="h-10 bg-slate-800 flex items-center px-4 shrink-0">
      <h1 className="text-white text-sm font-semibold">
        CBD Stone LLM — Clinical Decision Support
      </h1>
    </header>
    <div className="flex flex-col md:flex-row flex-1 min-h-0">
      {/* Mobile dashboard: above chat, hidden on desktop */}
      <div className="md:hidden shrink-0">
        <MobileDashboard />
      </div>
      {/* Chat panel */}
      <div className="w-full md:w-2/3 flex-1 md:flex-none md:border-r border-gray-200 min-h-0">
        <ChatPanel />
      </div>
      {/* Desktop dashboard: hidden on mobile */}
      <div className="hidden md:block w-1/3 bg-white">
        <DashboardPanel />
      </div>
    </div>
  </div>
</ChatProvider>
```

IMPORTANT: Do NOT modify any other files. DashboardPanel, ChatPanel, and all dashboard sub-components remain untouched. The desktop experience must be pixel-identical to the current layout.
